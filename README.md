# Deep Learning z PyTorch

### 🎯 Opis projektu
Implementacja i trenowanie głębokich sieci neuronowych (CNN) do rozpoznawania obiektów na zdjęciach.

### 🏗️ Szczegóły techniczne:
* **Architektura:** Wykorzystanie warstw konwolucyjnych (Conv2d), Pooling oraz warstw w pełni połączonych (Linear).
* **Optymalizacja:** Zastosowanie algorytmu Adam oraz funkcji straty CrossEntropyLoss.
* **Wyniki:** Model osiągnął satysfakcjonującą dokładność (accuracy), co potwierdzają wykresy przebiegu uczenia wewnątrz notebooka.

### 💡 Czego się nauczyłem:
* Zarządzania tensorami i procesem backpropagation.
* Wykorzystania GPU (CUDA) do przyspieszenia obliczeń.
* Zapobiegania przeuczeniu (Overfitting) poprzez odpowiednią architekturę.
